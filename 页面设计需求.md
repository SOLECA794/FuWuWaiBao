# 开发需求：学生端课堂学习页面重构
## 页面现状
当前页面结构：
1.  最左侧：固定的系统主导航栏（课堂学习、学习分析等菜单，保持不变，不要修改）
2.  中间主区域：顶部是「知识节点树」模块，下方是「知识状态/课堂交互/课堂笔记」三个标签页
3.  右下角：悬浮的AI问答按钮，点击才弹出对话面板
当前问题：功能零散、逻辑不耦合，问答功能和课堂学习内容完全割裂，布局松散，没有形成联动。

## 核心重构要求
### 1. 布局重构（核心，参考第二张参考图的左右分栏布局）
- 保留页面最左侧的系统主导航栏，**完全不动**
- 剩余页面空间，整体拆分为「左功能区 + 右问答区」左右两栏固定布局，比例为 6:4（左60%，右40%），两栏之间有分割线，支持拖拽调整宽度
- 左栏：收纳原页面的**全部原有功能**，包括知识节点树、知识状态、课堂交互、课堂笔记
- 右栏：**固定常驻的AI问答面板**，彻底删除右下角悬浮球设计，问答面板全程展示在页面右侧，和左栏学习内容完全联动

### 2. 左功能区设计（解决零散问题，用选项卡整合所有功能）
左栏整体采用「顶部选项卡 + 下方内容区」的结构，把所有零散功能用选项卡串联起来，和你之前设计的教师端页面设计语言统一：
1.  选项卡分为4个，固定在左栏顶部，分别是：
    - 「知识树」：对应原页面的知识节点树、学习进度模块
    - 「学习状态」：对应原页面的知识状态模块（已掌握/未掌握/前置节点卡片）
    - 「课堂交互」：对应原页面的课堂交互标签页内容
    - 「课堂笔记」：对应原页面的课堂笔记标签页内容
2.  点击选项卡，下方内容区对应切换，所有原有功能完整保留，只是用选项卡整合收纳，解决布局零散的问题
3.  所有模块保持卡片式设计，参考第二张参考图的精致卡片风格，圆角、轻微阴影、内边距统一，布局紧凑不松散

### 3. 右问答区设计（固定常驻，和左栏内容强耦合）
1.  右栏顶部固定标题：「AI课堂助手」，下方是完整的对话面板，和NextChat的对话界面风格统一，包含：
    - 对话历史滚动区，用户消息靠右，AI消息靠左，气泡样式
    - 底部固定输入框，支持Enter发送、Shift+Enter换行，发送按钮
2.  **核心联动逻辑（必须实现，解决逻辑割裂问题）**
    - 当用户在左栏切换「知识树」的知识点节点时，自动给对话面板发送系统提示：「当前学生正在学习【XX知识点】，请基于该知识点内容进行解答」，后续问答默认关联当前选中的知识点
    - 左栏「学习状态」里的未掌握知识点卡片，增加「问AI」按钮，点击自动跳转到右栏问答面板，同时自动填入问题：「请给我详细讲解一下【XX知识点】」
    - 左栏「课堂笔记」里的内容，增加「AI优化」按钮，点击自动把笔记内容发送到右栏，让AI优化润色笔记
3.  问答面板增加「上下文绑定」开关，默认开启，开启后所有问答都绑定当前课程的知识点内容，避免AI回答脱离课程

### 4. 风格与技术约束
1.  设计风格：和现有系统的浅色系、圆角卡片风格完全统一，同时参考第二张参考图的「紧凑精致、分栏清晰、模块分明」的设计感，不要松散空白
2.  技术栈：Vue3 + Element Plus，和现有项目技术栈统一，生成单文件.vue组件
3.  动画效果：选项卡切换用淡入淡出动画，卡片hover有轻微上浮效果，对话消息有流式打字机效果，和之前的设计统一
4.  响应式：页面宽度缩小时，右栏可折叠，点击按钮展开，不影响小屏使用

### 5. 输出要求
1.  先给重构后的布局设计思路，再给完整的.vue单文件组件代码
2.  代码有清晰注释，关键联动逻辑标注说明
3.  所有原有功能完整保留，不丢失任何现有能力
4.  代码可直接复制到现有项目中运行，无额外依赖

二、优秀作品的设计拆解 + 复刻方法
1. 这个页面的核心设计亮点（你要抄的核心）
这个页面是典型的 **「工作台分栏布局」**，也是现在 B 端产品、教育系统最主流的设计，核心优点就是「功能集中、逻辑清晰、不零散」：
左右分栏，主次分明：左栏是功能菜单 + 信息卡片，右栏是核心操作区（课表 / 任务列表），用户的视线从左到右，先看信息，再做操作，逻辑完全顺承
卡片化收纳，模块独立：所有零散的功能都用独立的圆角卡片装起来，每个卡片只做一件事，边界清晰，不会给人 “什么都有但什么都联系不起来” 的感觉
信息层级清晰：重要的内容放大、高亮，次要的内容缩小、收进卡片里，不会所有内容平铺，视觉上有重点
交互联动自然：左栏的信息和右栏的操作区强绑定，左栏选什么，右栏就对应展示什么，逻辑完全耦合
1. 1:1 复刻的步骤
先定死布局框架：用 CSS Flex 布局，把页面拆成「左功能栏 + 右操作栏」，比例固定，和你现在要改的学生端页面完全匹配
功能卡片化：把你原来平铺的「知识节点树、学习进度、已掌握 / 未掌握」，每个都做成独立的圆角卡片，加轻微阴影、统一内边距，和参考图的卡片风格对齐
用选项卡 / 标签页做收纳：把同一类的功能（比如学习相关的、笔记相关的）放进同一个选项卡里，不要全部平铺在页面上，解决零散问题
加联动逻辑：给每个卡片加操作按钮，点击后直接联动右侧的问答面板，比如 “未掌握知识点” 卡片加「问 AI」按钮，点击直接把问题填到右侧输入框，让功能之间产生联系
统一视觉细节：圆角统一用 8-12px，主色调用你系统的蓝色，字体大小分 3 级（标题 16px、正文 14px、辅助文字 12px），卡片间距统一用 16px，视觉上就会和参考图一样精致
