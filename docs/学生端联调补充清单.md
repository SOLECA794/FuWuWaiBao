# 学生端联调补充清单

更新时间：2026-03-31

## 已补齐的后端能力

- 个人中心返回结构已兼容前端 `camelCase` 字段，同时保留原有 `snake_case`。
- 收藏删除不再强制要求额外传 `studentId`。
- 复习计划会自动维护 `nextReviewDate`，并同步创建、更新、清理 reminder task。
- 练习判题与练习提交后，会自动更新知识图谱掌握度。
- 定时通知创建后，会自动同步生成对应调度任务。
- 通知、调度任务、任务状态接口已返回适合学生端直接使用的字段结构。

## 学生端新增可联调接口

### 个人中心

| 方法 | 路径 | 说明 |
|---|---|---|
| GET | `/api/v1/student/notes` | 获取学生笔记列表 |
| DELETE | `/api/v1/student/notes/{noteId}` | 删除笔记 |
| POST | `/api/v1/student/favorites` | 新增收藏或更新收藏标签 |
| GET | `/api/v1/student/favorites` | 获取收藏列表 |
| DELETE | `/api/v1/student/favorites/{favoriteId}` | 删除收藏 |

### 练习与错题

| 方法 | 路径 | 说明 |
|---|---|---|
| POST | `/api/v1/student/practice/generate` | 生成练习任务 |
| POST | `/api/v1/student/practice/submit` | 提交练习答案 |
| GET | `/api/v1/student/practice/history` | 获取练习历史 |
| GET | `/api/v1/student/practice/wrong-questions` | 获取错题列表 |
| POST | `/api/v1/student/practice/wrong-questions/{questionId}/retry` | 生成错题重做任务 |

### 复习计划

| 方法 | 路径 | 说明 |
|---|---|---|
| POST | `/api/v1/student/review-plans` | 创建复习计划 |
| GET | `/api/v1/student/review-plans` | 获取复习计划列表 |
| PUT | `/api/v1/student/review-plans/{planId}` | 更新复习计划 |
| DELETE | `/api/v1/student/review-plans/{planId}` | 删除复习计划 |
| POST | `/api/v1/student/review-plans/{planId}/items` | 添加复习项 |
| GET | `/api/v1/student/review-plans/{planId}/items` | 获取复习项列表 |
| PUT | `/api/v1/student/review-plan-items/{itemId}` | 更新复习项 |
| DELETE | `/api/v1/student/review-plan-items/{itemId}` | 删除复习项 |

### 知识图谱、通知、任务

| 方法 | 路径 | 说明 |
|---|---|---|
| GET | `/api/v1/student/knowledge-map` | 获取学生知识图谱 |
| GET | `/api/v1/student/knowledge-map/weak-points` | 获取薄弱知识点 |
| GET | `/api/v1/student/knowledge-map/strong-points` | 获取优势知识点 |
| GET | `/api/v1/student/knowledge-map/progress` | 获取学习进度分析 |
| GET | `/api/v1/student/knowledge-map/recommendations` | 获取学习推荐 |
| POST | `/api/v1/notifications` | 创建通知 |
| GET | `/api/v1/notifications` | 获取通知列表 |
| PUT | `/api/v1/notifications/{id}/read` | 标记单条通知已读 |
| PUT | `/api/v1/notifications/read-all` | 标记全部通知已读 |
| GET | `/api/v1/notifications/unread-count` | 获取未读通知数 |
| DELETE | `/api/v1/notifications/{id}` | 删除通知 |
| GET | `/api/v1/tasks/scheduled` | 获取调度任务列表 |
| POST | `/api/v1/tasks/scheduled/{id}/execute` | 立即执行任务 |
| DELETE | `/api/v1/tasks/scheduled/{id}` | 删除调度任务 |
| GET | `/api/v1/tasks/statuses` | 获取任务执行状态列表 |

## 学生端页面当前已接入

- 个人中心：笔记、收藏、复习计划、练习记录、错题重做、通知中心、任务中心。
- 通知中心：支持查看通知、查看未读数、单条已读、全部已读、删除通知。
- 任务中心：支持查看调度任务、立即执行任务、删除任务、查看执行状态。
